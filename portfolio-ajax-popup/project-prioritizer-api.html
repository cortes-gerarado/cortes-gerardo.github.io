Prioritizer API

  • Languages : Python, JS, PostgreSQL, HTML5
  • Skills : Database, APIs, Web development, Cloud Computing

The project showcases the design and builds of a relational database, authentication, and access control for the API and deploy of a Flask-based web application to the cloud.

This API helps to prioritize tasks by using an abstraction of the Eisenhower Matrix. Each assignment is classified as urgent or important to encourage a course of action.
